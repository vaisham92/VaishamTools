BrahMos

BrahMos is a performance analyzer tool for jmeter run performance tests.
This tool can be used to store all the performance logs obtained after the jmeter scripts are run. BrahMos tool uses the data present in the log file to generate graphs such as Area graph and Pie chart graph.
